# Veyoff

Privacy-enhanced client for Veyon-style screen monitoring systems.

## What is this?

Veyoff provides privacy controls for users being monitored by Veyon (or similar screen monitoring software). It gives the monitored user control over what the master sees.

## Features (Planned)

1. **Screen Freeze Hotkey** - Freeze your screen from master's perspective while continuing to work
2. **Master Presence Overlay** - Visual indicator when master is viewing your screen
3. **Selective Window Hiding** - Hide specific programs from master's view
